# spotilite-swing

Music Player like Spotify but Lite Desktop App.

## Features
1. Songs
- CRUD
- upload mp3/wav
- play, stop, prev, next song
2. Playlists
- CRUD
- add/remove song
3. Users
- Auth(Login, Signup)

## Tech Stacks
- Netbeans IDE
- Java OOP
- Java Swing
